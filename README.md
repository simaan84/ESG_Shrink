# Balancing Returns and Responsibility: Evidence from Shrinkage-Based Portfolios (R Code)

This repository contains R code to reproduce the analyses in:

**Makridis, Christos and Simaan, Majeed (2025).**  
*Balancing Returns and Responsibility: Evidence from Shrinkage-Based Portfolios.* SSRN.  
DOI: 10.2139/ssrn.4597152

> Abstract (short): We integrate ESG scores into portfolio selection and evaluate whether variance reductions can offset bias introduced by ESG data, using a large sample over 20 years and comparing return-only vs. ESG-based shrinkage rules.

SSRN page: https://ssrn.com/abstract=4597152
